|Key|Function|
|---|--------|
|![](/assets/images/calculator/key-c.png)|Clear key: clearing all calculating registers, except for the memory register.|
|![](/assets/images/calculator/key-ce.png)|Clear entry key: clear the immediate entry (only)|
|![](/assets/images/calculator/key-delta.png)|Delta percentage key: obtain an increase or decrease ratio (change in percentages). For example, sales for last year were $10,000. And for this year they are $12,000. Using this key, you can determine that there was a 20% increase in sales.|
|![](/assets/images/calculator/key-div.png)|Division key|
|![](/assets/images/calculator/key-dot.png)|Decimal point key|
|![](/assets/images/calculator/key-hash.png)|Hash key: non-add key, used to print numbers not part of the calculation. The diamond key displays subtotals|
|![](/assets/images/calculator/key-k.png)|Konstant key, used for performing multiplication and division by a constant number. With multiplication the first number is the constant. with division the second number is the constant. |
|![](/assets/images/calculator/key-mb.png)|Memory plus equal key: used for obtaining a calculation result (for multiplication and division only) and adding the result automatically to memory. This key is also used to add a number directly to memory.|
|![](/assets/images/calculator/key-mt.png)|Memory minus equal key (yellow): Used for obtaining a result (for multiplication and division only) and subtracting the result automatically from memory. When a negative number Is entered in minus memory, the result is a positive entry.|
|![](/assets/images/calculator/key-min.png)|Minus key|
|![](/assets/images/calculator/key-mu.png)|Mark up/down: Used for performing mark up {profit) and mark down calculations.|
|![](/assets/images/calculator/key-pf.png)|Paper feed|
|![](/assets/images/calculator/key-plus.png)|Plus key|
|![](/assets/images/calculator/key-pm.png)|Plus/minus key: Used for performing percentage calculations. Convenient for Add-on and Discount calculations. For example: 200 x 20% gives you 40.00, then if you press the "+" (plus key), you will get 240.00. This saves the time of having to manually add 40.00 to 200.00. The same works in reverse using the "-" (minus key). 200 x 20% gives you 40.00, then press the "-" (minus key), you will get 160.00.|
|![](/assets/images/calculator/key-s.png)|Memory subtotal key: Provides a temporary total for the contents of the memory.|
|![](/assets/images/calculator/key-star.png)|Total/equals key: Used to obtain a calculation result.|
|![](/assets/images/calculator/key-t.png)|Memory total key: Provides a Grand Total of what is in the memory.|
|![](/assets/images/calculator/key-x.png)|Multiply key|
|![](/assets/images/calculator/slider-am.png)|Accumulation switch: When this switch is set at the AM position, entries or calculation results are automatically accumulated In the memory every time the"*/=" key is pressed (also known as summation).
