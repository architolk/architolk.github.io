## Homepage

![](/assets/images/radio/events.png){:width="300px"}

At the homepage ([http://radiomarco.local](http://radiomarco.local)), information about the current radio station and song is depicted. It also presents the status of the volume, treble and bass dials.

Some extra information about the internet stations is displayed in the "informatie" box, as received by the internet radio stations. If the WiFi is interrupted or some other error occurs, this is displayed in this box. New information will only be displayed *after* the webpage has been loaded.

## Radio stations

![](/assets/images/radio/stations.png){:width="300px"}

At [http://radiomarco.local/stations](http://radiomarco.local/stations) you can find the current list of all radio stations that are configured. You can add, delete or edit these radio stations via this webpage. Status updates will be displayed below the row of buttons.

After you have made a change, these changes will directly be available at the radio (using the tuning dial or preset tuning dials). However, the information will only be stored in persistent memory after pressing the <Save> button.
